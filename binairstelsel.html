Binairstelsel

Het binaire stelsel bestaat uit slechts twee getallen. Het woord binair zegt het eigenlijk al. BI betekent "twee". Zo bestaat het binairstelsel uit slechts twee getallen namelijk de 1 en de 0.Deze 1 en 0 worden ook wel een bit genoemd. En een reeks van 8 1tjes en 0tjes wordt ook wel een byte genoemd. Een halfvolle pagina aan tekst wordt een KiloByte genoemd. Dat zijn dan 1.000 bytes.En een nummer van 1 minuut lang op je mp3 speler bevat 1MB ofwel 1 MegaByte. ASCII- tabel

ASCII is de afkorting voor American Standard Code for Information Interchange. Het ascii tabel is een code waarbij je kunt zien hoe elk karakter, letters cijfers leestekens etc. worden genoteerd in computertaal. Het wordt in decimaal, hexadecimaal, octaal en dus ook binair genoteerd. Om het in mensentaal uit te leggen: het is geheel vaan codes van de vele verschillende karaktertekens (cijfers, letters, leestekens en stuurcodes) zoals M door een nummer tussen 0 en 127. De ASCII code voor M bijvoorbeeld is 77. ASCII is de afkorting voor American Standard Code for Information Interchange. Om iets om te zetten in ASCII taal hoef je alleen maar in de tabel te zoeken naar het omzetten in het juiste karakter.

Hoeveel ascii codes bestaan er?

De normale ascii codes gaan van 0 tot 127. Als je het binairstelsel begrepen hebt begrijp je ook het volgende, deze 128 ascii codes worden samengesteld door een 7-bits code. Als je daar 8 van maakt kunnen er verschillende tekens en letters worden toegevoegd. Zo is het Ascii tabel ontstaan en en bestaat die nu uit wel 256 tekens!
